## Getting Started

1. **Prerequisites**
   - Flutter SDK (3.35.3 or higher)
   - Dart SDK
   - Android Studio / VS Code
   - Android device or emulator

2. **Installation**
   ```bash
   cd kultivate_app
   flutter pub get
   flutter run
   ```

3. **Running the App**
   ```bash
   flutter run
   ```
